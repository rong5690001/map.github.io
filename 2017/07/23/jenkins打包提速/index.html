jenkins打包提速

公司项目用jenkins打包，经过多次调整打包速度已经比之前快了很多，最近发现还有提速的空间，因为看到打包的log带了很多测试模块的东西（不需要打包的东西），所以就问了下度娘发现，只要在命令前面加上模块的名字（例如：clean -> app:clean）就可以对指定模块执行命令而不是对整个项目执行命令。重新修改命令，打包时间缩短了三分之一。
